I have now been in Boulder, CO for a month, interning at Unidata, a division of the University Corporation for Atmospheric Research. My primary task is developing a METeorological Aerodrome Reports (METAR) parser in Python which can be implemented into MetPy, a Python library for
meteorological applications. METARs are text reports that contain surface observation data,
including temperature, dewpoint, cloud cover, wind direction and speed, along with other variables.
They can be encoded by computers and humans, which can be problematic when it comes to consistency
in parsing.
